HTML

Ich bin eine Textauszeichnungssprache für Dokumente im WWW und bilde zusammen mit dem hypertext transfer protocol die Basis des Web's

HTML-Dokumente besteht aus verschiedensten Content z.B Text, Graphic, Video, Audio und den Steuerinformationen(die sog. TAG'S)

TAGS... sind Steuerinformationen für Web-Browser(Kennzeichung eines Bestimmten Abschnittes, Zeilenvorschub, hervorheben einer Textstelle

Einige Tags können auch Attribute enthalten. Wobei wiederum ein Attribut oftmals auch konkrete Attribute zugeordnet bekommen kann

Grundsätzliche Bestandteile eines Html Dokuments

Durch den einleitenden Start-Tag < HTML > wird dem WWW-Server mitgeteilt, dass es sich im Folgenden um ein HTML Dokument handelt. Der End Tag < /HTML > steht ganz am Ende

Das Dokument selbst besteht aus den Abschnitten < HEAD > ... < /HEAD > < BODY > ... < /BODY >

Der < HEAD > abschnitt enthält allgemeine Iformationen über das Dokument. Hierzu gehört der Abschnitt < title > ... < / title>, durch den der Dokumenten titel festgelegt wird

der Inthalt des Anschließenden < BODY > Tag wird im fensterinneren des Web-Browsers dargestellt. Der < BODY > tag kann u.a auch Attribute enthalten, wie Folgt:

BACKGROUND="name.jpg", BGCOLOR=', Text="# rr gg bb", Link= (Hypertext Link Farbe), VLINK(angeblicher link-Farbe), ALINK(aktiver link farbe)

Strukturierte Darstellung von Aufzählungen, listen

Zu unterscheiden sind
Ungeordnete Listen < ul> . . . < /ul> , werden Markierungssymbole benutzt
Geordnete Listen < ol> . . . < /ol>, werden durchnummeriert

Listenelemente innerhalb beider Listentypen mittel < li> eingeleitet. Sie werden in der Regel Eingerückt und mit einem vorangestellten Symbol dargestellt.

  • Listen können auch ineinander Geschachtelt werden
  • Attribute für < ul > sind
    • type = "disc"
    • type = "circle"
    • type = "square"
  • Attribute für < ol>
    • Type = 1, A, a, i, I
    • start = n
  • Attribute für < li>
    • Type = disc, circle, square, 1, A, a, i, I
    • value = n

    Grafiken(Images ) unter HTML

    werden mittels < img src="Pfadangabe"> eingebunden zb. < img src="NTA-Logo.gif">

    • Grafiken, die nur in einem Speziellen Dokument verwendet werden sollten in selben Verzeichnis wie die HTML-Dokument selbst abgelegt werden
    • Grafiken, die von vielen Dokumenten verwendet werden, sollten nur einmal an zentraler Stelle gespeichert werden, und mit einem Absoluten Pfad referenziert werden
    • weitere Atribute im Image Tag < img>:
      • ALT = "Alternativ text" Alternativtext
      • TITLE = "Text" Text,der immer Angezeigt wird
      • Align = "top, middle, bottom"
        • top - ist bündig mit oberen Rand der Textzeile
        • middle - Grafik ist mittenzentriert zur Baseline Text
        • bottom - Grafik ist bündig mit grundebene des Textes
      • Width = "Pixelbreite der Grafik"
      • Height = " Pixelbreite der Grafik"
      • Border = " n" Rahmen der Grafik in Pixel
      • Vspace = " n " Vertikaler Abstand in Pixel
      • Hspace = "n" Vertikaler Abstand in Pixel
      • LEFT / RIGHT = " Mehrzeiliger Text neben der Grafik" < br> bewirkt einen Vorschub in der nächsten Zeile neben der Grafik.
        Zeilenvorschub in die nächste Zeile unterhalb der Grafik erfolgt mit < br clear="All"

    Können an belibiger Stelle im HTML Dokument stehen

    Sprungziele innerhlb eines Dokuments können folgendermaßen realisiert werden

    • Festlegen eines Links < A HREF="#Sprungziel">Text < /a>
    • Sprungstelle (bookmark) im Text markiert mit < A NAME="Sprungziel"> < /A>
    • Springe auf Bookmark in anderen Dokumenten < A HREF="URL #bookmark"> bookmark Text < /a>
    • Link als Grafik < A HREF="Ziel URL"> < img src="Pfad/Bilddatei"> < /a>
      Grafik erhält ggf. blauer Ramen
    • Mail - Link < A HREF="mailto: Mailaddresse"> link-Text < /A>
      Kann ein Mail Link eingefügt werden. Damit öffnet sich ein Fensterzur eingabe der Mail, in dem die Zieladresse bereits eingetragen ist
